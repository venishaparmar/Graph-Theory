Real Life Application

Application of different graphs

Application

Computer networks

In computer science and network science, network theory is a part of graph theory: a network can be defined as a graph in which nodes and/or edges have attributes

Application

Medical Science

There are several biological domains where graph theory techniques are applied for knowledge extraction from data.

Application

Transportation

Graphs are used to model situation in which a commodity is transported from one location to another. Highway systems can be thought of as transporting cars.

Application

Social Media

A graph is made up of nodes; just like that a social media is a kind of a social network, where each person or organization represents a node. These nodes in a social media are interdependent on each other

Application

Math and Research

Graph theory uses a set of mathematical principles and formulas to examine the relationships among objects of interest

Application

Satellite Communication

In satellite Communication, space craft, space shuttle, planets can consider as nodes and the path in which they are situated and rotate can be consider as edge.
